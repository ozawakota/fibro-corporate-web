2024年03月25日
「ゼリージュース・イサゴール」原料の紅麹色素についていつもフィブロ製薬をご愛顧頂き誠にありがとうございます。
先般ニュースで報道がございました、「紅麹」原料を使用した製品において健康被害が発生した件に関しまして、弊社製品「ゼリージュースイサゴール」で使用しております「紅麹色素」はヤヱガキ醗酵技研社製を使用しており、小林製薬社製「紅麹」とは一切関係がなく、同メーカーから紅麴原料の供給は受けておりません。
また、紅麹色素については日本においては厚生労働省によって既存添加物(着色料)に認可されていますので健康被害についての情報はございませんことをご報告申し上げます。

どのようなことでお困りですか？

「自社ブランドの商品をつくりたい」「商品企画やアイデアはあるがどこに生産を頼めばよいか分からない」
「より機能を持たせたい」「食物繊維を使った健康食品をつくりたい」「OEMでどんなものがつくれるか知りたい」など、
OEMで困っていることがあれば、是非、フィブロ製薬株式会社にご相談ください。
弊社ではお客様のご要望を丁寧にヒアリングし、ベストな解決策をご提案します。

フィブロ製薬では、お客様のご予算、ご要望に合わせた製品づくりの
ご提案を行うなど、お客様のニーズにお応えしています。

フィブロ製薬の製品づくりは、安全性、機能性及び品質を重要視することはもちろん、
さらに“健康的においしく食すこと”をモットーに常に力を入れています。

OEMを依頼したいとお考えの方をフィブロ製薬は全力で応援いたします。
お客様の満足につながればとの想いで、
徹底した”こだわり”で、モノづくりを行っています。

健康食品OEMをお考えのお客様、まずはフィブロ製薬にご相談ください！

豊富な経験・知識を持ったスタッフが、お客様の「こんな製品を作りたい」という想いをカタチにします。
